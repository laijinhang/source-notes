在net包里dial.go文件minNonzeroTime方法看到IsZero方法，time.Time类型：IsZero方法，判断time.Time类型变量是否为零值（也就是是否为默认值）

我以前判断time.Time是否为默认值
```go
var t time.Time
var timeNil time.Time
if t == timeNil {
    return ""
}
```
新的方式
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	t := time.Date(0001, 1, 1, 00, 00, 00, 00, time.UTC)
	fmt.Printf("%v\n", t.IsZero())	// true
	t = time.Date(0002, 1, 1, 00, 00, 00, 00, time.UTC)
	fmt.Printf("%v\n", t.IsZero())	// false
}
```
